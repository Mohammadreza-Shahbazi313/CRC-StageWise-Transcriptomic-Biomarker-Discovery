# CRC-StageWise-Transcriptomic-Biomarker-Discovery

An interpretable machine-learning framework for stage-specific transcriptomic biomarker discovery in colorectal cancer progression using TCGA COAD and READ RNA-seq datasets.

## Project Overview

This repository contains the computational workflow developed for identifying progression-associated biomarkers across sequential colorectal cancer stages (Stage I–II, Stage II–III, and Stage III–IV).

The pipeline includes:

* RNA-seq preprocessing and normalization
* Feature selection
* Machine-learning benchmarking
* Feature-importance filtering
* Consensus biomarker prioritization
* Biological interpretation

Additional scripts, processed outputs, documentation, and reproducibility resources will be added following manuscript acceptance.
